# GlassOS Assets Directory
This directory contains all graphical assets for GlassOS.

## Structure

```
assets/
├── icons/          # Application and system icons
├── fonts/          # Custom fonts (optional)
├── images/         # Background images and graphics
└── sounds/         # System sounds (optional)
```

## Icons

Place PNG icons here for applications:
- `start.png` - Start menu button icon
- `browser.png` - AeroBrowser icon
- `notepad.png` - GlassPad icon
- `calculator.png` - Calculator icon
- `explorer.png` - AeroExplorer icon
- `weather.png` - Weather app icon
- `settings.png` - Settings icon

Recommended size: 48x48 pixels with transparency.
